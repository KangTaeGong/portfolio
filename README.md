## 📌 Intro

많은 사람의 불편한 점을 해소해 주는 서비스를 만드는 것이 삶의 목표입니다.<br>
독학으로 공부하면서 많이 돌아가기도 하고, 시간도 오래 걸렸지만, 아직은 꿈을 이루기 위해 배우는게 재밌는 취준생입니다.<br>
인프런과 구글링, 유튜브 미디어를 통해 공부를 하고, 프로젝트도 혼자 진행해 보면서 많은 것을 배울 수 있었습니다.<br>
